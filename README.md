# Prueba Antigravity

Este es el repositorio **Prueba_Antigravity** creado por Antigravity.

## Descripción

Repositorio de prueba para demostrar la integración de Antigravity con GitHub.

## Storybook

Este proyecto utiliza Storybook para desarrollar y documentar componentes de UI de forma aislada.

### Instalación

Antes de iniciar, asegúrate de instalar las dependencias:

```bash
npm install
```

### Ejecutar Storybook

Para iniciar Storybook en modo de desarrollo:

```bash
npm run storybook
```

Esto abrirá Storybook en tu navegador predeterminado (normalmente en `http://localhost:6006`).
